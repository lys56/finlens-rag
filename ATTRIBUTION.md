# Attribution

FinLens RAG 是在既有金融文档 RAG 原型基础上的个人二次开发版本，并非从零独立实现。

本次可明确归属的改动包括：

- 将模型服务密钥改为仅从环境变量读取，提供 `.env.example`；
- 增加按来源文件筛选的检索能力；
- 增加可下载的 Markdown 对话与来源记录；
- 重写 GitHub README、运行说明与发布忽略规则；
- 移除示例数据、向量库与任何本地密钥，整理为可公开发布的仓库结构。

发布前请补充原作者/原仓库链接（若有）及双方认可的贡献说明，并始终保留原项目许可证。简历和面试中应如实说明这是基于原型的二次开发及自己负责的模块。
